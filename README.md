# 100 GAMES

## Play

Open `index.html` in a browser (requires a local server — see below) to see the game picker.
Click any game to launch it. While playing, tap the **≡ menu** (top-right) to switch games.

### Run locally

```bash
python3 -m http.server 8000
# then open http://localhost:8000
```

### Deploy

Push to GitHub and enable **GitHub Pages** (Settings → Pages → Deploy from `main`).
No build step, no dependencies.
